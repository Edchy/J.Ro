## Design Context

### Users
Conscious coffee buyers and curious locals — people who care about provenance and slow craft. They arrive via word of mouth or Instagram, not search ads. They read. They look. They're not in a hurry. The site is a brochure: it shouldn't sell, it should convince through atmosphere.

### Brand Personality
**Warm, rooted, slow.** Järna Rosteri is Sweden's only biodynamic roastery — a small, human-scale operation that has been roasting by hand every week since 2013. The brand has earned its understated confidence. It doesn't shout. It doesn't perform craft — it simply is.

### Aesthetic Direction
- **Tone**: Quiet authority. Like a good book cover — every element earns its place.
- **Theme**: Dark (JR Grön as primary background) with warm ochre/gold accents — not cold-luxury white, not coffee-shop brown. The green is the brand's defining colour; lean into it.
- **Typography**: Maax Mono for all running text — monospaced but editorial, not "developer". Orator Std only for the wordmark and logotype instances.
- **Motion**: Purposeful and unhurried. The existing parallax and scroll-driven animations set the bar — extend that register.

### Anti-References
- **Not**: Kraft paper / hipster roastery / reclaimed wood aesthetics
- **Not**: Cold luxury — no maximalist white space with gold serifs
- **Not**: Generic e-commerce — no product grids, no conversion CTA stacks
- **Not**: Startup / tech — no Inter, no gradient buttons, no social proof

### Design Principles
1. **Restraint over decoration** — Every added element must earn its place. Prefer removing over adding.
2. **Warmth through material, not ornament** — Colour, texture, and scale create feeling. Not illustration, not icons, not badges.
3. **Monospace as editorial** — Maax Mono is not a code font here. Use it with generous leading and careful sizing to achieve literary rhythm.
4. **Motion is atmosphere** — Animations should feel like the roastery itself: deliberate, unhurried, slightly heavy. No snappy or bouncy easing.
5. **One thing remembered** — Each section should leave one impression, not several. Edit until only the essential remains.
